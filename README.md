This is a single-file web application. It means that all the ingredients needed to make the calculator look good and function properly are packed into one single .html file. You don't need to install anything or be connected to the internet to use it—your web browser (like Chrome, Safari, or Edge) reads the code and runs it instantly.

The 3 Main Ingredients
The code is split into three layers, like a cake:

1. HTML (The Bones / Structure): This creates the actual pieces on the screen. It builds the display box where your numbers show up, and defines all the buttons (0-9, +, -, *, /, C, and =).

2. CSS (The Clothes / Styling): Without CSS, the calculator would just be a messy list of buttons on a white screen. The CSS steps in to center it perfectly on your screen, add the modern dark background, color-code the operators orange, and make the buttons react with a smooth change in color when you hover your mouse over them.

3. JavaScript (The Brains / Logic): This is what makes the calculator actually work. When you click a button, JavaScript takes that number and types it onto the screen.
